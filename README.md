# Kiyim-Kechak Frontend

React frontend for wholesale clothing CRM system.

## Ishga tushirish

### Avtomatik (Windows):
\`\`\`bash
start_frontend.bat
\`\`\`

### Manual:
\`\`\`bash
# 1. Dependencies
npm install

# 2. Server
npm start
\`\`\`

## API URL sozlamalari

`.env.local` faylida API URL ni o'zgartiring:
\`\`\`
REACT_APP_API_URL=http://localhost:8000/api
\`\`\`

## Features

- ✅ Responsive design
- ✅ Dashboard with statistics
- ✅ Customer management
- ✅ Product management
- ✅ Sales management
- ✅ Search and filters
- ✅ Toast notifications
- ✅ Loading states
- ✅ Modal dialogs

## Tech Stack

- React 18
- React Router v6
- Axios for API calls
- React Hot Toast for notifications
- Lucide React for icons
- CSS3 with custom styling

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
